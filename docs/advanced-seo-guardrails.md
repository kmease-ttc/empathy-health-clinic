# Advanced SEO Guardrails

Purpose: Enforce high-impact technical, content, performance, and
authority rules that maximize long-term organic growth.

------------------------------------------------------------------------

## Site Architecture

-   Important pages within 2--3 clicks of homepage
-   Hub-and-spoke topic clusters enforced
-   Prevent infinite filter/facet URLs
-   No crawl traps

------------------------------------------------------------------------

## Index Quality Control

-   Auto-noindex:
    -   thin tag pages
    -   internal search results
    -   low-value utility pages
-   Consolidate similar content with canonical + redirects

------------------------------------------------------------------------

## URL Normalization

-   HTTPS only
-   lowercase URLs
-   single host (www or non-www)
-   consistent trailing slash
-   strip tracking parameters
-   max one redirect hop

------------------------------------------------------------------------

## Topical Authority

-   One pillar page per core topic (2k--5k words)
-   5--20 interlinked support articles per pillar
-   tight internal linking between cluster content

------------------------------------------------------------------------

## Content Freshness

-   review content every 6--12 months
-   update stats, screenshots, year references
-   resubmit updated URLs in sitemap

------------------------------------------------------------------------

## Search Intent Matching

-   informational pages = blogs/guides
-   transactional pages = product/services
-   no intent mixing

------------------------------------------------------------------------

## SERP Optimization

-   keyword front-loaded titles
-   modifiers: best, guide, checklist, free, 2026
-   rich result schema whenever possible
-   snippet-friendly formatting (lists, tables, short answers)

------------------------------------------------------------------------

## Internal Link Equity

-   new content auto-links to pillar + related posts
-   old high-traffic pages link to new content
-   varied anchor text (not exact match only)

------------------------------------------------------------------------

## Performance Budgets

-   HTML \< 100kb
-   JS \< 300kb target
-   compressed modern images
-   lazy loading below fold
-   CWV must pass before publish

------------------------------------------------------------------------

## UX Signals

-   no intrusive popups on load
-   engagement-based CTAs
-   clean mobile experience

------------------------------------------------------------------------

## Authority & Backlinks

-   monitor spam links
-   disavow when needed
-   build linkable assets:
    -   tools
    -   research
    -   stats posts
    -   resources

------------------------------------------------------------------------

## Entity & AI-Era SEO

-   consistent entity references
-   link to authoritative sources
-   full topic graph coverage
-   answer-first content layout

------------------------------------------------------------------------

## CI Fail Conditions

-   orphan pages
-   thin content
-   broken internal links
-   CWV failures
-   duplicate intent pages
-   poor architecture depth

------------------------------------------------------------------------

Goal: Build a fast, authoritative, tightly structured site Google trusts
as a topical leader.
