# SEO Best Practices Checklist

## Page Structure
- Exactly one H1 per page
- H2/H3 used in logical hierarchy
- Title tag between 50–60 characters
- Meta description between 140–160 characters
- URL is short, readable, and keyword-focused
- No duplicate titles or meta descriptions

## Content Quality
- Content answers a clear user intent
- Primary keyword appears in:
  - Title tag
  - H1
  - First 100 words
  - One H2
- Secondary keywords used naturally
- No keyword stuffing
- Content written for humans first

## Internal Linking
- Every page links to at least one other page
- Anchor text is descriptive (not “click here”)
- Important pages receive more internal links
- No orphan pages

## Technical SEO
- HTTPS enabled
- Canonical tag present on every page
- XML sitemap exists and is submitted
- robots.txt allows indexing of public pages
- No broken internal links
- No redirect chains

## Images
- Every image has descriptive alt text
- Image file names are readable (not IMG_1234)
- Images are compressed
- No images larger than needed for display
- Use modern formats where possible (WebP)

## Performance
- Page loads in under 2.5s on mobile
- Minified CSS and JS
- Lazy-load non-critical images
- Avoid render-blocking scripts
- Use caching headers

## Mobile
- Mobile-first layout
- Text readable without zoom
- Buttons large enough to tap
- No horizontal scrolling

## Indexing & Monitoring
- Google Search Console connected
- Google Analytics (or GA4) installed
- Sitemap regularly updated
- Index coverage errors reviewed monthly
