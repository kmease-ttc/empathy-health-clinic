# Blog SEO Optimization Rules

Purpose: Ensure every blog post is structurally optimized for maximum
organic visibility, crawlability, and engagement.

------------------------------------------------------------------------

## Content Length & Depth

-   Target 1,200--2,500 words for competitive topics
-   Minimum 800 words for any indexable blog post
-   Cover topic comprehensively (definitions, examples, FAQs, next
    steps)
-   Include original insights or data when possible

------------------------------------------------------------------------

## Title (H1)

-   50--60 characters preferred
-   Include primary keyword near the front
-   Clear benefit or intent match
-   One H1 only per page

Examples: - How to Fix Crawl Errors (Step‑by‑Step Guide) - Local SEO
Checklist for Service Businesses

------------------------------------------------------------------------

## Meta Title

-   55--60 characters
-   Primary keyword + modifier (guide, checklist, tools, 2026, etc.)
-   Unique across site

------------------------------------------------------------------------

## Meta Description

-   140--160 characters
-   Include keyword naturally
-   Clear CTA or value promise

------------------------------------------------------------------------

## Headings Structure

-   One H1
-   Use H2s for major sections
-   Use H3s for subpoints
-   Include related keywords in headings naturally

------------------------------------------------------------------------

## Internal Linking (Critical)

Minimum: - 3--5 internal links pointing OUT to relevant posts/pages -
2--3 internal links pointing IN from other posts (via hub strategy)

Rules: - Use descriptive anchor text (not "click here") - Link to: -
cornerstone content - service/product pages where relevant - related
blogs

------------------------------------------------------------------------

## URL Structure

-   Short and keyword focused
-   Use hyphens
-   No dates unless news-focused

Example: /blog/technical-seo-checklist

------------------------------------------------------------------------

## Images

-   At least 1 image per 300--500 words
-   Descriptive alt text with keywords where natural
-   Compressed for speed
-   Use modern formats when possible

------------------------------------------------------------------------

## Keyword Usage (Natural)

-   Primary keyword in:
    -   title
    -   H1
    -   first 100 words
    -   at least one H2
    -   meta title/description
-   Use semantic variations throughout

Avoid: - stuffing - forced repetition

------------------------------------------------------------------------

## Featured Snippet Optimization

Include when relevant: - numbered steps - bullet lists - short
definitions (40--60 words) - tables

------------------------------------------------------------------------

## Readability

-   Short paragraphs (2--4 lines)
-   Bullet lists
-   Clear subheadings
-   Conversational tone

Target: - Grade 6--9 reading level

------------------------------------------------------------------------

## E-E-A-T Signals

-   Author name + bio
-   Updated date
-   Real examples or experience
-   Cite reputable sources when relevant

------------------------------------------------------------------------

## Schema

Use where applicable: - Article - FAQ - HowTo - Breadcrumb

Ensure visible content matches schema

------------------------------------------------------------------------

## Engagement Signals

-   Table of contents for long posts
-   Clear next step CTA
-   Related posts section

------------------------------------------------------------------------

## Indexing & Technical

-   Self-referencing canonical
-   Indexable (no noindex)
-   Included in sitemap
-   Loads fast (CWV friendly)

------------------------------------------------------------------------

## CI Fail Rules for Blogs

Fail publish if: - Word count \< 800 - Missing H1 - Missing meta title
or description - No internal links - Duplicate title - Broken images -
Non-200 status - No canonical

------------------------------------------------------------------------

## Gold Standard Blog Checklist

-   Comprehensive topic coverage
-   Optimized title + meta
-   Strong internal linking
-   Clear structure
-   Fast loading
-   Rich media
-   Schema applied
-   Valuable to reader

------------------------------------------------------------------------

Goal: Every blog post should be a ranking asset --- not filler content.
