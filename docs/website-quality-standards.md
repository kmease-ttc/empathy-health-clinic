# Website Quality Standards

## Accessibility
- All images have alt text
- Color contrast meets WCAG AA
- Forms have labels
- Pages usable with keyboard only
- No critical info conveyed by color alone

## UX & Clarity
- Clear primary CTA on every page
- Contact info easy to find
- Navigation consistent across pages
- Headings summarize content accurately
- No dark patterns or misleading UI

## Content Governance
- Content reviewed before publishing
- No placeholder or lorem ipsum text
- Dates updated when content changes
- Medical or legal claims are sourced
- No outdated references

## Security
- No exposed API keys
- No client secrets in frontend code
- Forms protected against spam
- Dependencies kept up to date

## SEO Hygiene
- No duplicate pages
- No thin content pages
- No auto-generated filler content
- No hidden text or links
