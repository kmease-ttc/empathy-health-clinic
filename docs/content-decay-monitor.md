# Content Decay Monitor

## Purpose
Detect pages losing visibility and identify when refreshes are required.

## Signals Monitored
- Impressions
- Clicks
- Average position
- Keyword coverage
- SERP competitor displacement

## Decay Detection
- Sustained downward trends
- Ranking replacement by competitors
- Stale content signals
- Missing coverage vs competitors

## Output
- Prioritized refresh candidates
- Likely decay reasons
- Recommended refresh actions:
  - Update sections
  - Expand topical coverage
  - Add FAQs or schema
  - Improve internal links

## Usage Rule
Content decay triggers refresh workflows.
