# Backlink & Authority Signals Service

## Purpose
Track authority signals and explain ranking changes.

## Signals Tracked
- New and lost backlinks
- Referring domains
- Anchor text distribution
- Link velocity
- Authority proxies

## Competitive Comparison
- Authority vs competitors
- Link gap analysis

## Output
- Authority health summary
- Root cause classification:
  - Page weakness vs authority weakness
- Link-building priorities

## Usage Rule
Authority drops trigger outreach or PR actions.
